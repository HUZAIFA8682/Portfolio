Project Overview

This project is an end-to-end data automation pipeline built to identify, evaluate, and outreach to relevant companies at scale. It automates the complete workflow — from raw company data ingestion to intelligent filtering and controlled email outreach — while remaining resume-safe, fault-tolerant, and production-ready.

The system is designed with real-world constraints in mind, including rate limits, interruptions, unreliable websites, and data quality issues.

Problem It Solves

Manually finding company websites, checking relevance, shortlisting leads, and sending outreach emails is slow, error-prone, and does not scale.

This system replaces manual work with a reliable automation pipeline capable of processing thousands of companies with minimal human intervention.

How the System Works

1️⃣ Data Ingestion

The pipeline starts with a raw CSV of company records (e.g., Companies House data).

2️⃣ Web Scraping Engine

Using Selenium, the system searches for official company websites, extracts contact emails, mimics human browsing behavior with random delays, and avoids reprocessing already scraped companies. The process is resume-safe and can be stopped and restarted without losing progress.

3️⃣ Intelligent Filtering & Scoring

Each website is analyzed using a regex-based keyword scoring engine. Positive keywords increase relevance, negative keywords discard irrelevant companies, and weighted rules determine final categorization.

Companies are automatically classified into: Best Match, Good Match, Potential Match, or Discard, with results exported into structured Excel reports.

4️⃣ Email Outreach Automation

A secure SMTP-based outreach system validates emails, enforces daily sending limits, introduces random delays, prevents duplicate emails, and uses Gmail App Password authentication for security.

5️⃣ API Layer (Optional)

A FastAPI backend allows CSV uploads, background scraping job execution, and result downloads, enabling easy integration with dashboards or frontends.

Key Technical Features

  • Resume-safe execution with automatic progress backups
  • Fault-tolerant design handling missing sites and failures
  • Regex-driven keyword decision engine
  • Human-like scraping behavior
  • CSV-first, transparent data pipeline
  • Modular architecture (scraper → filter → outreach)
  • Production-aware constraints and validations

Tech Stack

Python · Selenium · Pandas · FastAPI · SMTP (Gmail App Passwords) · Regex-based NLP · Excel Automation (openpyxl)
