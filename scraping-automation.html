How the System Works

The pipeline moves data through five distinct stages, ensuring only high-quality leads receive outreach.

1. Data Ingestion

Starts with a raw CSV of company records (e.g., Companies House data).

2. Scraping Engine

Selenium searches for websites, extracts emails, and mimics human behavior.

3. Filtering & Scoring

Regex-based scoring classifies companies as Best Match, Potential, or Discard.

4. Email Outreach

SMTP system validates emails and sends campaigns with limits and delays.

5. API Layer

FastAPI backend for uploading CSVs and triggering background jobs.

Under the Hood

The intelligent scoring engine uses NLP-style keyword density to classify leads automatically. 


def score_company(website_text, keywords):
    score = 0
    matches = []
    
    # Regex-based keyword density analysis
    for word, weight in keywords.items():
        if re.search(rf"\b{word}\b", website_text, re.I):
            score += weight
            matches.append(word)
    
    # Lead Classification Logic
    if score >= 85:
        return "TIER_1_LEAD", matches
    elif score >= 50:
        return "TIER_2_POTENTIAL", matches
    else:
        return "DISCARD", []
